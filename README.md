# Maximus Segurança

**Maximus Segurança** é um site desenvolvido para fornecer soluções de segurança tanto para residências quanto para empresas, com foco em tecnologia e inovação, proporcionando uma experiência online simples e eficiente.

## Tecnologias Utilizadas

- **HTML**: Estruturação do conteúdo da página.
- **CSS**: Estilos e layout responsivo para uma experiência visual atraente.
- **JavaScript**: Funcionalidades interativas e dinâmicas.
- **Pixel do Facebook**: Implementação para rastreamento e análise de métricas de desempenho e marketing.

## Funcionalidades

- **Design Responsivo**: O site é otimizado para funcionar em diversos dispositivos e tamanhos de tela.
- **Integração com o Pixel do Facebook**: Permite o rastreamento das ações dos usuários para otimizar campanhas de marketing e gerar relatórios de desempenho.

## Como Rodar o Projeto

1. Clone este repositório:

   ```bash
   git clone https://github.com/seu-usuario/maximus-seguranca.git

   ```

2. Navegue até a pasta do projeto:

   ```bash
   cd maximus-seguranca
   ```

3. Abra o arquivo `index.html` em seu navegador para visualizar o site.
